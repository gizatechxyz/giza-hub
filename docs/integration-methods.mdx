## Comparison

| Feature | Agentic | IaaS (Intelligence) |
|---------|---------|---------------------|
| **What Giza Provides** | Complete solution:<br/>Smart accounts + Execution + Optimization | Optimization intelligence only |
| **What You Manage** | User interface + monitoring |  Execution infrastructure |
| **Transaction Execution** | Automatic by Giza agents | You execute based on optimizer output |

| **Best For** | Partners without execution infrastructure | Partners with existing execution infrastructure |

## Agentic Integration

### What is it?

The **Agentic approach** uses the Giza Agent SDK to integrate fully autonomous yield optimization agents. Agents automatically manage capital, rebalance across protocols, and optimize for maximum APR without any manual intervention.

**Giza manages everything:**
- Self-custodied smart account creation ([ZeroDev](https://zerodev.app/))
- Session key management
- Transaction execution
- Gas payment
- Continuous optimization
- Protocol rebalancing

**You manage:**
- User interface
- Deposit flow
- Performance display
- Withdrawal requests

### When to Use

<AccordionGroup>
  <Accordion title="You want a complete solution">
    Perfect if you want to offer yield optimization without building execution infrastructure.
  </Accordion>
  <Accordion title="You don't have smart account infrastructure">
    Giza provides smart accounts, session keys, and execution - you don't need to build this.
  </Accordion>
  <Accordion title="You need fast time-to-market">
    Integrate in hours, not weeks. The SDK provides everything out of the box.
  </Accordion>
</AccordionGroup>

## IaaS Integration (Intelligence as a Service)

### What is it?

The **Intelligence as a Service (IaaS)** approach provides access to Giza's **Optimizer** as a stateless intelligence service. You already have your own execution infrastructure (smart accounts, transaction execution, capital management, or even Vaults), and you want to leverage Giza's optimization intelligence to inform your rebalancing decisions.

**Giza provides:**
- Optimization intelligence (optimal allocations)
- APR improvement calculations
- Detailed action plans
- Execution-ready calldata
- Stateless API

**You manage:**
- Your own execution infrastructure
- Rebalancing schedule
- Capital management

### When to Use

<AccordionGroup>
  <Accordion title="You have existing execution infrastructure">
    Perfect if you already have smart accounts, transaction execution, or even Vaults, and just need optimization intelligence.
  </Accordion>
  <Accordion title="You need custom execution logic">
    Build your own rebalancing schedule, risk management, or execution strategies while leveraging Giza's optimizer.
  </Accordion>
  <Accordion title="You want full control">
    Decide when to rebalance, which suggestions to execute, and how to manage risk - optimizer just provides intelligence.
  </Accordion>
  <Accordion title="You want the optimization brain, not the execution">
    Use Giza's AI-powered optimization engine without handing over execution to agents.
  </Accordion>
</AccordionGroup>
