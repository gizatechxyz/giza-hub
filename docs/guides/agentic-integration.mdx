## Overview

The **Agentic integration** is the full-service option. Giza handles smart account creation, execution infrastructure, gas management, and continuous optimization.

<Check>
**Choose Agentic if:**
- You don't have your own execution infrastructure
- You want yield optimization without manual management
- You need to ship quickly
- Your users want hands-off yield generation
</Check>

## What Giza Manages

- **Smart Account Creation** - Self-custodied accounts via ZeroDev
- **Session Key Management** - Secure delegated permissions
- **Transaction Execution** - All rebalancing operations
- **Gas Payment** - Users don't pay gas for agent operations
- **Continuous Optimization** - 24/7 APR monitoring and rebalancing

## What You Build

- User interface for deposits and withdrawals
- Performance dashboards
- Notification systems
- User authentication
