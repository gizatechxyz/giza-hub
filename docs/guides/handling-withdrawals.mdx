## Overview

How to implement partial and full withdrawals for users.

<Info>
These methods are called on an `Agent` handle. Get one via `giza.createAgent(eoa)`, `giza.getAgent(eoa)`, or `giza.agent(wallet)`.
</Info>

## Partial Withdrawal

Agent remains active, only withdraw specified amount:

```typescript
const result = await agent.withdraw('500000000'); // 500 USDC (6 decimals)

if ('total_value' in result) {
  console.log(`Withdrawn: ${result.total_value_in_usd} USD`);
  console.log('Details:', result.withdraw_details);
}
```

## Full Withdrawal

Deactivate agent and withdraw everything:

```typescript
// Initiate full withdrawal (deactivation)
await agent.withdraw();

// Poll for completion
const finalStatus = await agent.waitForDeactivation({
  interval: 5000,
  timeout: 300000,
  onUpdate: (status) => {
    updateUI(status); // Update progress in UI
  }
});

console.log('Withdrawal complete:', finalStatus.status);
```

## Error Handling

```typescript
try {
  await agent.withdraw();
} catch (error) {
  if (error instanceof TimeoutError) {
    // Still polling, funds safe
    showMessage('Withdrawal taking longer than expected...');
  } else if (error instanceof GizaAPIError) {
    // Check agent status
    const currentStatus = await agent.status();
    if (currentStatus.status === 'deactivating') {
      // Still in progress
      showMessage('Withdrawal in progress, please wait...');
    }
  }
}
```
